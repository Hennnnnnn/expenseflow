# ExpenseFlow

ExpenseFlow is an AI-powered personal finance platform built with Go.

## Features

* Gmail transaction import
* BCA email parser
* Telegram Bot
* SQLite / PostgreSQL
* AI transaction categorization
* Monthly financial reports
* REST API
* Docker support

## Tech Stack

* Go
* Chi
* SQLite
* PostgreSQL
* sqlc
* golang-migrate
* MailKit equivalent (go-imap)
* Telegram Bot API
* OpenAI API
* Docker
